GET /api/asset/ 获取待采集主机列表

说明

客户端（SSH / Salt 模式）调用此接口获取需要采集资产信息的主机列表， 然后通过 POST /api/asset/ 上报采集结果。

当前返回： {% for h in host_list %} {{ h }} {% endfor %}
客户端请求请加 ?raw=1 或设置 Accept: application/json 获取纯 JSON。
POST /api/asset/ 上报资产采集结果

鉴权

请求需带 query 参数：key 和 ctime
生成方式：key = md5("alkdjwqm|" + ctime)，ctime 为当前时间戳（秒）
有效期 2 秒，且 key 不可复用（防重放）

请求头

名称值
Content-Typeapplication/json

Body 参数（JSON）

字段类型必填说明
action string 是 操作类型：create（新主机首次上报）/ update（更新）/ update_host（主机名变更）
basic object 是 基本信息（主机名 / 系统 / SN / 制造商 / 型号 / CPU）
main_board object 否 主板信息（SN / 制造商 / 型号）
cpu object 是 CPU 信息（物理颗数 / 逻辑颗数 / 型号）
disk object 否 硬盘信息，key 为槽位（slot），value 为硬盘详情
memory object 否 内存信息，key 为槽位（slot），value 为内存详情
nic object 否 网卡信息，key 为网卡名（name），value 为网卡详情
old_hostname string 否 仅 action=update_host 时需要，旧主机名

请求示例

# 客户端 Python 示例
import requests, json, hashlib, time

ctime = time.time()
key = hashlib.md5("alkdjwqm|{}".format(ctime).encode()).hexdigest()

payload = {
    "action": "create",
    "basic": {"data": {"hostname": "web01", "os_platform": "linux", "os_version": "CentOS 7"}},
    "main_board": {"data": {"sn": "xxxxx", "manufacturer": "Dell", "model": "R730"}},
    "cpu": {"data": {"cpu_count": 8, "cpu_physical_count": 1, "cpu_model": "Intel Xeon"}},
    "disk": {"data": {
        "0": {"slot": "0", "pd_type": "SAS", "capacity": 279.396, "model": "SEAGATE"}
    }},
    "memory": {"data": {
        "DIMM1": {"slot": "DIMM1", "capacity": 8, "manufacturer": "Samsung", "model": "DDR4", "sn": "xxx", "speed": "2400"}
    }},
    "nic": {"data": {
        "eth0": {"hwaddr": "aa:bb:cc:dd:ee:ff", "netmask": "255.255.255.0", "ipaddrs": "192.168.1.10", "up": True}
    }}
}

resp = requests.post(
    "http://127.0.0.1:8001/api/asset/",
    params={"key": key, "ctime": ctime},
    data=json.dumps(payload),
    headers={"Content-Type": "application/json"}
)
print(resp.json())
